CSC 86030 - Proposal

I propose a real-time algorithm that reconstructs a 3D model of a corridor using a single image from a mobile device. My approach will use a non-iterative, simultaneous model reconstruction method called J-Linkage, which is both accurate in parameter estimation and efficient in computation. I will first use J-Linkage to find the vanishing points in a scene and then use the detected vanishing points along with corresponding straight line edges to accurately determine the important features of a corridor. The algorithm will take generate multiple models over time and each will be combined. A 3D map of the corridor will be the result with all images being taken into consideration. I will use a genetic algorithm to determine the best parameters for use in the general algorithm. Furthermore, using the more confident information from previous and closer data to adjust what parameters should be used in the rest of the corridor using DDDS. The ability to model 3D representations of corridors in real-time on a mobile device can be used in a range of applications including mobile apps to help the visually impaired navigate, robotic navigation, building inspection, and virtual tours.
